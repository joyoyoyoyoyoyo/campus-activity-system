# 简化进度计划

> 基于 WBS 中的关键活动（A1–A8）制定的轻量级进度安排，作为周度对齐的依据。

## 1. 时间基准

- 项目起始：2026-04-15（待小组确认）
- 计划周期：约 8 周
- 时间单位：周（W1–W8）

## 2. 关键里程碑

| 里程碑 | 时间   | 标志事件                                       |
| ------ | ------ | ---------------------------------------------- |
| M1     | W1 末  | 需求与 WBS 定稿，仓库与目录就绪                |
| M2     | W3 末  | 架构与数据模型确定，主线模块脚手架完成         |
| M3     | W5 末  | 报名主线（FR-01/02/03）开发联调通过            |
| M4     | W6 末  | 签到与统计（FR-04/05）开发联调通过             |
| M5     | W7 末  | 功能验收与缺陷收敛                             |
| M6     | W8 末  | 总结报告完成、main 分支整理完毕、最终交付      |

## 4. 依赖关系（简化）

```
A1 需求 ─▶ A2 架构 ─▶ A3 报名 ─▶ A4 签到 ─▶ A5 统计

```

## 5. 风险与应对

| 风险                     | 触发条件             | 应对                                       |
| ------------------------ | -------------------- | ------------------------------------------ |
