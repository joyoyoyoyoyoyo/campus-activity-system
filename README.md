# 校园活动报名与签到管理系统

> 软件项目管理实验项目（软工23级）
> 学院：计算机学院　|　指导教师：佟强　|　适用专业：软件工程
> 编写日期：2026.4.15

## 一、项目简介

学校希望建设一套轻量级的"校园活动报名与签到管理系统"，支持活动信息发布、学生报名、名单审核、二维码签到和签到统计导出。系统面向校内活动组织场景，强调流程清楚、记录完整和使用便捷。

## 二、项目目标

- 支持活动管理人员发布活动并维护报名时间、人数上限和活动地点。
- 支持学生查看活动并完成在线报名。
- 支持活动组织者对报名名单进行审核与查看。
- 支持现场二维码签到，并形成签到结果统计。
- 支持活动结束后导出基本统计结果。

## 三、主要使用角色

| 角色           | 职责                                                 |
| -------------- | ---------------------------------------------------- |
| 活动管理员     | 发布活动、维护活动信息、查看报名与签到统计           |
| 学生           | 查看活动、提交报名、查询个人报名结果                 |
| 现场签到人员   | 核验学生身份并完成签到记录                           |
| 系统维护人员   | 维护系统参数与基础配置                               |

## 四、功能范围

### 纳入范围

- 活动信息发布与维护
- 学生报名信息提交
- 报名名单查看与审核
- 签到记录与签到状态管理
- 签到统计结果导出

### 不纳入范围

- 复杂支付功能
- 与真实校园统一身份认证系统的深度对接
- 高并发优化与正式部署方案
- 移动端 App 正式开发
